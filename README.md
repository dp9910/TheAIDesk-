# The AI Desk

**[aiyellowpage.online](https://www.aiyellowpage.online/)**

Powerful AI tools ship every week but most of them get lost in the noise. The people who would benefit most, researchers, faculty, small businesses, never hear about them.

The AI Desk closes that gap:

- **Weekly one-pagers** that break down one AI tool at a time. What it does, how to use it, why it matters. No jargon.
- **Subscribe** to get each new issue delivered to your inbox via Google Forms.
- **Book a free consultation** through Cal.com for hands-on guidance tailored to your workflow or business needs.

## Tech Stack

Next.js 16 (App Router, TypeScript), Tailwind CSS v4, Vercel, Cal.com, Google Forms

## Running Locally

```bash
cd website && npm install && npm run dev
```
